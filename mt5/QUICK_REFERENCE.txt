
```
Entry:  EMA(20) > EMA(50) + RSI < 70
Exit:   +100 pips TP / -50 pips SL
Hours:  12:00-16:00 UTC (London-NY overlap)
Risk:   2% per trade, Max 3 concurrent
```

### Setup
```bash

python scripts/mt5_alpaca_bridge.py \
  --config config/trading_config.json \
  --key YOUR_ALPACA_KEY \
  --secret YOUR_ALPACA_SECRET
```
