# Plaid Fix - Quick Reference Card

## 🚨 The Problem
```
error_code: "INVALID_API_KEYS"
error_message: "invalid client_id or secret provided"
```

## ✅ The Solution

### In 3 Steps:

**Step 1: Get Real Credentials**
```
→ Go to: https://dashboard.plaid.com/
→ Click: Settings → API Keys
→ Copy: Client ID and Secret
```

**Step 2: Update .env File**
```
File: C:\Users\winst\BentleyBudgetBot\.env

Find:
PLAID_CLIENT_ID=677f5a06bfb57e001d2ca8e9
PLAID_SECRET=7ce1f012fdd48c25854e3ecc5d1a91

Replace with:
PLAID_CLIENT_ID=<your_real_client_id>
PLAID_SECRET=<your_real_secret>
```

**Step 3: Verify**
```bash
cd C:\Users\winst\BentleyBudgetBot
python test_plaid_credentials.py
```

Expected: `✅ ALL TESTS PASSED!`

## ❌ Common Mistakes

| Mistake | Fix |
|---------|-----|
| Credentials too short | Copy exact value from dashboard |
| Extra spaces/quotes | Remove spaces before & after |
| Wrong environment | Use "sandbox" for testing |
| Old/expired credentials | Get fresh ones from dashboard |
| Didn't restart app | Restart Streamlit after updating |
