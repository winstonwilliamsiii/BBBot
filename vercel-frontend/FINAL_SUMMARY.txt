## 🚀 3-STEP DEPLOYMENT

### Step 1: Get API Key (2 minutes)
```
1. Go to: https://cloud.appwrite.io/console/project-68869ef500017ca73772/settings/api-keys
2. Copy your Server API Key
3. Keep it safe (NEVER commit to git)
```

### Step 2: Add to Vercel (3 minutes)
```
1. Go to: https://vercel.com/dashboard/bentley-budget-bot
2. Project Settings → Environment Variables
3. Add all NEXT_PUBLIC_* variables from .env.local.example
4. Add APPWRITE_API_KEY (the one you copied)
5. Save!
```

### Step 3: Deploy (1 minute)
```bash
git add .
git commit -m "Vercel frontend with secure API routes"
git push origin main
# Vercel auto-deploys! ✨
```

**Total Time: ~6 minutes to production!**
